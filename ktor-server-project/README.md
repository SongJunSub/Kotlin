# Ktor 서버 프로젝트

이 프로젝트는 코틀린 기반의 경량 웹 프레임워크인 Ktor를 사용하여 구축한 샘플 서버 애플리케이션입니다.

## 시작하기

프로젝트를 실행하고 테스트하려면 다음 단계를 따르세요.

### 요구사항

- Java 11 또는 그 이상
- Gradle

### 실행

다음 명령어를 사용하여 애플리케이션을 실행할 수 있습니다.

```bash
./gradlew run
```

서버는 `http://localhost:8080`에서 실행됩니다.

### 테스트

다음 명령어를 사용하여 테스트를 실행할 수 있습니다.

```bash
./gradlew test
```
